## Objective
To identify and exploit SQL Injection vulnerabilities using a vulnerable web application.

## Tools Used
- Kali Linux
- DVWA (Damn Vulnerable Web Application)
- Web Browser

## Description
SQL Injection vulnerability was tested in the DVWA SQL Injection module. 
Manual SQL Injection was successfully performed by manipulating the `id` parameter, 
allowing unauthorized access to database records.

## Impact
- Exposure of sensitive database information
- Unauthorized data access

## Mitigation
- Use prepared statements
- Input validation and sanitization
- Least privilege database access

## Result
Successfully demonstrated SQL Injection attack in DVWA.
